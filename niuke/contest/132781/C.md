# C. 运动会积分统计

## 题目描述

哈尔滨华德学院举办了一场为期 $n$ 天的运动会，第 $i$ 天产生的积分值为 $a_i$ （可能为负数，表示扣分）。  
运动会结束后，裁判组需要回答 $q$ 次询问，每次询问给出两个整数 $l, r$ ，要求计算从第 $l$ 天到第 $r$ 天（包含两端）的积分总和。  
  


## 输入描述

第一行包含两个整数 $n, q$ （ $1 \le n, q \le 10^5$ ）。  
第二行包含 $n$ 个整数 $a_i$ （ $|a_i| \le 10^9$ ），表示每天的积分。  
接下来 $q$ 行，每行两个整数 $l, r$ （ $1 \le l \le r \le n$ ），表示询问的起止天数。  


## 输出描述

对于每个询问，输出一行一个整数，表示该时间段的积分总和。  


## 样例

### 样例 1
**输入**:
```
5 3
1 2 3 4 5
1 3
2 5
1 1
```

**输出**:
```
6
14
1
```
