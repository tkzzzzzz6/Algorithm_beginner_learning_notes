# F. 最优通勤路线

## 题目描述

小QAQ工作在一座大城市，城市中有 $n$ 个公交站点，站点编号 $1 \sim n$ ，站点之间有 $m$ 条公交线路，每条线路连接两个站点，乘坐一次需要花费一定的时间（单位：分钟）。所有线路都是**双向**的。  

小QAQ每天从公司站点 $s$ 出发，先要去银行站点 $x$ 取钱，然后再去餐厅站点 $t$ 吃午饭。他想知道：从公司出发，经过银行，最后到达餐厅的最短总耗时是多少？  

注意：  

路径必须依次经过 $s \to x \to t$ ，但中间可以经过其他站点。（**没说不能原地不动 哦 QWQ**）  

保证站点之间是连通的，不存在无法到达的情况。  

**可能存在重边或自环。**  


## 输入描述

第一行包含三个整数 $n, m$  $(1 \le n \le 10^5, 1 \le m \le 2 \times 10^5)$ ，分别表示站点数和公交线路数。  

接下来 $m$ 行，每行三个整数 $u, v, w$  $(1 \le u, v \le n, 1 \le w \le 10^9)$ ，表示站点 $u$ 和 $v$ 之间有一条耗时 $w$ 分钟的公交线路。  

最后一行包含三个整数 $s, x, t$  $(1 \le s, x, t \le n)$ ，分别表示公司站点、银行站点、餐厅站点。  


## 输出描述

输出一个整数，表示从 $s$ 出发经过 $x$ 到达 $t$ 的最短总耗时。  


## 样例

### 样例 1
**输入**:
```
4 4
1 2 3
2 3 4
3 4 5
1 4 10
1 3 4
```

**输出**:
```
12
```

**说明**:  

从 1 到 3 最短路径为 1→2→3，耗时 3+4=7。  
从 3 到 4 最短路径为 3→4，耗时 5。  
总耗时 7+5=12。  


### 样例 2
**输入**:
```
3 2
1 2 1
2 3 2
1 1 3
```

**输出**:
```
3
```

**说明**:  

公司即银行（s = x = 1），因此从 1 到 1 耗时为 0；再从 1 到 3 的最短路径为 1→2→3，耗时 1+2=3，总耗时 3。  


### 样例 3
**输入**:
```
3 2
1 2 5
2 3 10
2 2 2
```

**输出**:
```
0
```

**说明**:  

****公司、银行、餐厅均为同一个站点 2（s = x = t = 2）。不需要乘坐任何公交，总耗时为 0
