# E. 军训物资分配

## 题目描述

哈尔滨华德学院军训期间，教官需要为 $n$ 件物资分配背包空间。每件物资有重量 $w_i$ 和价值 $v_i$ 。背包总负重为 $W$ 。每件物资最多只能选择一件（即 0/1 选择）。请计算在不超过背包负重的前提下，能带走物资的最大总价值。  


## 输入描述

第一行包含两个整数 $n, W$ （ $1 \le n \le 10^3$ ， $1 \le W \le 10^5$ ）。  
接下来 $n$ 行，每行两个整数 $w_i, v_i$ （ $1 \le w_i \le 10^5$ ， $1 \le v_i \le 10^9$ ）。  


## 输出描述

一个整数，表示能获得的最大总价值。  


## 样例

### 样例 1
**输入**:
```
3 5
2 3
3 4
4 5
```

**输出**:
```
7
```
