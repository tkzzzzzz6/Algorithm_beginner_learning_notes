# I. 不好，Alice和Bob又来做游戏了

## 题目描述


Alice 和 Bob 在一棵有 $n$ 个节点的树上进行游戏，游戏由 Alice 先手。  

给定一个节点 $r$ 作为**树的根**。游戏开始时，将一枚棋子放在根节点 $r$ 上。随后双方轮流移动棋子：**每次移动必须将棋子从当前节点移动到它在有根树意义下的某个子节点上**。无法移动（即当前节点为叶子节点）的一方判负。  

双方均采取最优策略。  

对于每个节点 $r = 1, 2, \dots, n$ ，请你判断：**若以 $r$ 为根独立进行游戏，先手 Alice 是否必胜**。  
  


## 输入描述

第一行包含一个整数 $n$ （ $1 \le n \le 2 \times 10^5$ ），表示树的节点数。  

接下来 $n-1$ 行，每行包含两个整数 $u, v$ ，表示树上的一条边。  


## 输出描述

输出 $n$ 行，第 $i$ 行对应以 $i$ 为根时的游戏结果：若 Alice 必胜，输出 Alice；否则输出 Bob（不含引号，区分大小写）。  


## 样例

### 样例 1
**输入**:
```
5
1 2
2 3
2 4
4 5
```

**输出**:
```
Bob
Alice
Bob
Alice
Alice
```
