## 题目大意

$n$ 个同学，每人至少刷一题。已知：
- 刷 A、B、C 题的人数分别为 $a, b, c$
- 恰好只刷两题的有 $d$ 人
- 求三道题都刷的人数 $ans$

## 公式推导

根据容斥原理：
$$n = a + b + c - (d + 2 \times ans)$$

> 解释：$a+b+c$ 把刷一题的算1次，两题的算2次，三题的算3次。需要减去重复部分。

整理得：
$$ans = \frac{(a + b + c) - n - d}{2}$$

## 代码

```cpp
#include <iostream>
#define il inline
#define endl '\n'
using namespace std;

#define fastio ios::sync_with_stdio(false); cin.tie(0);

il void solve(){
    int n, a, b, c, d;
    cin >> n >> a >> b >> c >> d;
    cout << ((a + b + c) - n - d) / 2 << endl;
}

int main(){
    fastio;
    int t; cin >> t;
    while(t--) solve();
    return 0;
}
```

**时间复杂度**：$O(t)$，$t$ 为测试数据组数
