字符串和数据类算法思路上是比较相似的

原地操作

方法1:双指针

不需要分类讨论字符串长度的奇偶,如果是奇数,最中间的直接跳过

for循环解法
```cpp
/*

class Solution {
public:
  void reverseString(vector<char> &s) {
    int n = s.size();
    for(int i = 0,j = n - 1;i<n/2;++i,--j){
        swap(s[i],s[j]);
    }
  }
};

```

while循环解法(这个更好理解一点)
```cpp
/*
 * @Author: tkzzzzzz6
 * @Date: 2026-04-12 14:43:32
 * @LastEditors: tkzzzzzz6
 * @LastEditTime: 2026-04-12 14:43:59
 */
class Solution {
public:
  void reverseString(vector<char> &s) {
    int n = s.size();
    int left = 0,right = n -1;
    while(left < right){
        swap(s[left],s[right]);
        ++left;
        --right;
    }
  }
};
```
