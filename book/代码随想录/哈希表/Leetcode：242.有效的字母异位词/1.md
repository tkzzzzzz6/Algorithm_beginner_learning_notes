题目说明,判断两个字符串是不是用相同的字符组成的

哈希对应的数据结构可以分成三类

1. 数组
2. set
3. map

方法1:数组哈希

具体思路:定义一个数组,长度为26,每个元素初始值为0,遍历字符串s,每次遇到一个字符就把对应的数组元素加1,遍历字符串t,每次遇到一个字符就把对应的数组元素减1,最后判断数组中是否有不为0的元素,如果有的话说明s和t不是异位词,如果没有的话说明s和t是异位词
>可以提前判断s和t的长度,如果长度不相同的话,直接返回false




```cpp
class Solution {
public:
  bool isAnagram(string s, string t) {
    int hash[26] = {0};
    if (s.size() != t.size()) // 如果s和t的长度不相同的话,不可能是异位词
      return false;

    for (auto c : s) {
      hash[c - 'a']++;
    }
    for (auto c : t) {
      hash[c - 'a']--;
    }
    for (int i = 0; i < 26; ++i) {
      if (hash[i] != 0)
        return false;
    }
    return true;
  }
};

```
