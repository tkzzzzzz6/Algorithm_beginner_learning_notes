遍历
- 深度优先遍历:一般使用递归的方式实现,或者使用栈实现
  - 前序:中左右
  - 中序:左中右
  - 后序:左右中
- 广度优先遍历:一般使用迭代的方式实现,也就是队列实现
  - 层序遍历

二叉树的定义方式

```cpp
struct TreeNode {
    int val;
    TreeNode *left;
    TreeNode *right;
    TreeNode(int x):val(x),left(nullptr),right(nullptr){};
}
```
