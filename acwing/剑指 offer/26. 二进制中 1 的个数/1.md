计算机中数的二进制表示采用的是补码

正数:
    1. 最高位为0
    2. 其他位为数的二进制表示
负数:
    1. 最高位为1
    2. 其他位为数的二进制表示取反后加1

一个数的补码+本身,等于一个开头为 1 后面面全为0的数,这个数叫做模,模等于2的n次方,其中n是二进制数的位数

cpp 中负数转化为无符号数时,会将负数的补码当做无符号数的二进制表示,所以会得到一个很大的数

```cpp

* 26. 二进制中1的个数

// @acwing code start

class Solution {
public:
    int NumberOf1(int n) {
        unsigned int us_n = n;
        int s = 0;
        
        while(us_n)s += us_n & 1,us_n >>= 1;
        
        return s;
    }
};
// @acwing code end
```
