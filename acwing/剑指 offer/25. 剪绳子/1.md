思路:判断 尽可能拆分多个 3的情况下对应的乘积是最大的

推断:当 n>=5时，尽可能拆分多个 3的情况下对应的乘积是最大的

推导过程:
1. 当 n >= 5是,n = 3 + (n-3) 乘积为 3 * (n-3) > n


情况 1:n%3==0 直接拆分成 n/3个 3
情况 2:n%3==1 先拆分成 n/3-1个 3 和 4
情况 3:n%3==2 先拆分成 n/3个 3 和 2

// @acwing code start

class Solution {
public:
    int maxProductAfterCutting(int length) {
        if(length <= 3)return 1 * (length-1);

        int res = 1;

        if(length % 3 == 1)length-=4,res = 4;
        else if(length % 3 == 2)length-=2,res =2;
        
        while(length)res*=3,length-=3;
        return res;
    }
};
// @acwing code end
