

可以使用 cpp 中的与运算的短路时间 if 和 while等分支判断的功能,并使用;作为空运算,不操作

# 代码

写法 1:
```cpp
class Solution {
   public:
    int getSum(int n) {
        int t = 0;
        (n > 0) && (t = n + getSum(n - 1));
        return t;
    }
};
```


写法2:
再利用 cpp 中的逗号表达式的性质

```cpp
class Solution {
   public:
    int getSum(int n) {
        int res = n;
        return ((n>0) && (res += getSum(n-1)),res);
    }
};
```
