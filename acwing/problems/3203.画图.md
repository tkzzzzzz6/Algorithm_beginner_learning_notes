
# 2. 画图

## 问题描述

给定 $n$ 个矩形，每个矩形由左下角和右上角的坐标确定。统计这些矩形覆盖的格点数量。

### 输入格式

- 第一行：整数 $n$
- 接下来 $n$ 行：每行四个整数 $x_1, y_1, x_2, y_2$，表示矩形的左下角和右上角坐标

### 输出格式

输出覆盖的格点数量。

### 数据范围

- $1 \le n \le 100$
- $0 \le x_1 < x_2 \le 100$
- $0 \le y_1 < y_2 \le 100$

### 输入样例

```
2
1 1 4 4
2 3 6 5
```
![1778570906524.png](https://tk-pichost-1325224430.cos.ap-chengdu.myqcloud.com/blog/1778570906524.png)

### 输出样例

```
15
```



## 思路:暴力枚举

遍历所有可能的格点，检查每个格点是否被至少一个矩形覆盖,若覆盖了就 break。

- 时间复杂度：$O(n \times 100^2)$
- 空间复杂度：$O(n)$



## 代码实现

```cpp
#include <iostream>
#include <cstring>
#include <algorithm>

using namespace std;
const int N = 101;
int a[N],b[N],c[N],d[N];
int x,y;

int main(){
    int n;cin >> n;
    int res = 0;
    for(int i = 1;i<=n;++i){
        cin >> a[i] >> b[i] >> c[i] >> d[i];
    }

    for(int x = 0;x<=100;++x)
        for(int y = 0;y<=100;++y)
            for(int i = 1;i<=n;++i){
                if(x >= a[i] && y >= b[i] && x+1 <= c[i] && y+1 <= d[i]){
                    res++;
                    break;
                }
            }

    cout << res << endl;

    return 0;
}
```
