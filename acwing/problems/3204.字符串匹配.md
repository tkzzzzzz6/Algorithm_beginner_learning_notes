# 3.字符串匹配

## 问题描述

给定一个字符串 $S$，在 $n$ 行文字中找出包含 $S$ 的所有行。程序需支持大小写敏感选项：
- 选项为 `1`：大小写敏感（'A' ≠ 'a'）
- 选项为 `0`：大小写不敏感（'A' = 'a'）

### 输入格式

- 第一行：字符串 $S$（由大小写英文字母组成）
- 第二行：整数 `type`（0 = 不敏感，1 = 敏感）
- 第三行：整数 $n$（文字行数）
- 接下来 $n$ 行：每行一个待查文字（由大小写英文字母组成，无空格）

### 输出格式

输出包含字符串 $S$ 的所有行，按输入顺序输出。

### 数据范围

- $1 \le n \le 100$
- 每个字符串长度：不超过 100

### 输入样例

```
Hello
1
5
HelloWorld
HiHiHelloHiHi
GrepIsAGreatTool
HELLO
HELLOisNOTHello
```

### 输出样例

```
HelloWorld
HiHiHelloHiHi
HELLOisNOTHello
```

### 样例说明

由于 `type = 1`（大小写敏感）：
- 第 1 行 `HelloWorld` 包含 `Hello` ✓
- 第 2 行 `HiHiHelloHiHi` 包含 `Hello` ✓
- 第 3 行 `GrepIsAGreatTool` 不包含 `Hello` ✗
- 第 4 行 `HELLO` 大小写不同 ✗
- 第 5 行 `HELLOisNOTHello` 包含 `Hello` ✓

若 `type = 0`（大小写不敏感），第 4 行也应输出。

## 思路 1:库函数

- 时间复杂度：$O(n \cdot m)$，其中 $n$ 是行数，$m$ 是每行的平均长度
- 空间复杂度：$O(n \cdot m)$（存储输入的字符串）

使用 C++ 字符串的 `find()` 函数进行匹配：
- **大小写敏感**：直接用 `find()` 查找
- **大小写不敏感**：先将两个字符串转换为小写，再用 `find()` 查找



```cpp
#include <algorithm>
#include <cstring>
#include <iostream>

using namespace std;

// 转换字符串为小写
string toLower(string s) {
    string result = "";
    for (auto c : s) {
        result += tolower(c);
    }
    return result;
}

int main() {
    string s;
    cin >> s;
    
    int type;
    cin >> type;
    
    int n;
    cin >> n;

    while (n--) {
        string line;
        cin >> line;
        
        if (type == 1) {
            // 大小写敏感
            if (line.find(s) != string::npos)
                cout << line << endl;
        } else {
            // 大小写不敏感：都转小写后比较
            if (toLower(line).find(toLower(s)) != string::npos)
                cout << line << endl;
        }
    }
    
    return 0;
}
```
