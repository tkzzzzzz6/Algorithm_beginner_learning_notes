# 相邻数对

## 问题描述

给定 $n$ 个不同的整数，统计其中有多少对整数的值正好相差 $1$。

## 输入格式

- 第一行：整数 $n$（给定整数的个数）
- 第二行：$n$ 个整数

## 输出格式

输出值相差 $1$ 的数对个数。

## 数据范围

- $1 \le n \le 1000$
- 整数范围：$[0, 10000]$ 的非负整数

## 输入样例

```
6
10 2 6 3 7 8
```

## 输出样例

```
3
```

## 样例说明

值相差 $1$ 的数对：$(2, 3), (6, 7), (7, 8)$，共 3 对。

## 思路:排序 + 遍历

1. 对数组排序
2. 遍历排序后的数组，统计相邻元素差值为 $1$ 的次数

- 时间复杂度：$O(n \log n)$（排序主导）
- 空间复杂度：$O(n)$

> notice
> 
> 题目保证所有输入整数都不相同，因此排序后只需检查相邻元素是否满足 `a[i] - a[i-1] == 1`，每次满足就计数加 1,如果是求数对个数的话对应的结果应该是所有满足条件的相邻元素对的个数即 `i*j`,其中 i 和 j 分别是两个相邻元素的出现次数。

## 代码实现

```cpp
#include <iostream>
#include <algorithm>

using namespace std;

const int N = 1010;
int a[N];

int main(){
    int n;
    cin >> n;
    for(int i = 1; i <= n; ++i)
        cin >> a[i];

    // 排序数组
    sort(a + 1, a + n + 1);

    // 统计相邻差值为 1 的个数
    int res = 0;
    for(int i = 2; i <= n; ++i){
        if(a[i] - a[i-1] == 1)
            res++;
    }
    
    cout << res << endl;

    return 0;
}
```
