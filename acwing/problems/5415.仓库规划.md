# 1. 仓库规划

## 题目描述

有 n 个仓库，每个仓库有 m 个位置编码。对于每个仓库 i，找出一个编号 j，使得 j != i 且仓库 j 的位置编码在每一维上都严格大于仓库 i 的位置编码。如果存在多个满足条件的仓库 j，输出编号较小的那个；如果不存在满足条件的仓库 j，输出 0。

## 思路:暴力枚举

- 时间复杂度：O($n^2*m$)
- 空间复杂度：O($n*m$)

枚举每个仓库 i，枚举其他仓库 j，判断仓库 j 的位置编码是否在每一维上都严格大于仓库 i 的位置编码。

如果有仓库 j 满足条件，直接 break, 输出其编号；否则输出 0。

### 代码


```cpp
#include <algorithm>
#include <cstring>
#include <iostream>

using namespace std;

const int N = 1010, M = 10; // N仓库个数上届,M位置编码维数上界
int w[N][M];                // 仓库的位置编码
int n, m;

int main() {
  scanf("%d%d", &n, &m);
  for (int i = 1; i <= n; ++i)
    for (int j = 0; j < m; ++j)
      scanf("%d", &w[i][j]);

  for (int i = 1; i <= n; ++i) {
    int id = 0;
    for (int j = 1; j <= n; ++j) {
      if (i == j)
        continue;
      bool is_bigger = true;
      for (int k = 0; k < m; ++k) {
        if (w[j][k] <= w[i][k]) {
          is_bigger = false;
          break;
        }
      }
      if (is_bigger) // 题目要求输出编号较小的仓库的编号即可
      {
        id = j;
        break;
      }
    }
    printf("%d\n", id);
  }

  return 0;
}
```
