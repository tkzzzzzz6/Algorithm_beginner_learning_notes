排序+滑动窗口
应为是两数的最小值,所以比的是窗口的两端

# 解法1

```py
'''
Author: tkzzzzzz6
Date: 2026-01-25 11:03:22
LastEditors: tkzzzzzz6
LastEditTime: 2026-01-25 11:03:26
'''
class Solution:
    def minimumDifference(self, nums: List[int], k: int) -> int:
        if k == 1:
            return 0
        nums.sort()
        n = len(nums)
        ans = float('inf')
        for i in range(n-k+1):
            ans = min(ans,nums[i+k-1]-nums[i])
        return ans
```

# 解法2(更好理解的解法)

```py
'''

LastEditors: tkzzzzzz6
LastEditTime: 2026-01-25 11:03:33
'''
class Solution:
    def minimumDifference(self, nums: List[int], k: int) -> int:
        if k == 1:
            return 0
        nums.sort()
        n = len(nums)
        l,r = 0 ,k-1
        ans = float('inf')
        while r < n:
            ans = min(ans,nums[r]-nums[l])
            l += 1
            r += 1
        return ans
```
