# 3070. 元素和小于等于 k 的子矩阵的数目

## 题意

给定 `m×n` 的非负整数矩阵 `grid` 和整数 `k`，统计**左上角为 (0,0)**、右下角为 `(i,j)`（0-indexed）的子矩阵中，元素之和 ≤ k 的子矩阵个数。

---

## 方法一：二维前缀和

### 核心思路

用一个 `(m+1)×(n+1)` 的 `sum` 数组保存二维前缀和，其中 `sum[i+1][j+1]` 表示 `grid[0..i][0..j]` 所有元素之和。

**为什么多一圈？**  
令第 0 行/列全为 0，这样在计算 `sum[i+1][j+1]` 时，无论 `i`、`j` 是否为 0，都可以直接访问 `sum[i][...]` 和 `sum[...][j]`，**无需单独判断边界**。

### 转移公式推导

```
sum[i+1][j+1] = sum[i][j+1]      // 上方矩形
              + sum[i+1][j]       // 左侧矩形
              - sum[i][j]         // 重复减去左上角（容斥）
              + grid[i][j]        // 加上当前格
```

### 计数方式

```cpp
ans += sum[i+1][j+1] <= k;
```

C++ 中比较运算符返回 `bool`（`true`=1，`false`=0），直接加到 `ans` 上，等价于 `if (...) ans++`，写法更简洁。

### 代码

```cpp
class Solution {
public:
    int countSubmatrices(vector<vector<int>>& grid, int k) {
        int m = grid.size(), n = grid[0].size();
        int ans = 0;
        vector<vector<int>> sum(m + 1, vector<int>(n + 1));

        for (int i = 0; i < m; ++i) {
            for (int j = 0; j < n; ++j) {
                sum[i+1][j+1] = sum[i][j+1] + sum[i+1][j] - sum[i][j] + grid[i][j];
                ans += sum[i+1][j+1] <= k;
            }
        }
        return ans;
    }
};
```

**时间复杂度** O(mn)，**空间复杂度** O(mn)。

---

## 拓展：原地二维前缀和

直接在 `grid` 上累加，省去额外空间。因为 `grid` 从 `(0,0)` 开始，需要用 `if` 判断是否存在上方/左侧邻居。

```cpp
class Solution {
public:
    int countSubmatrices(vector<vector<int>>& grid, int k) {
        int m = grid.size(), n = grid[0].size();
        int ans = 0;

        for (int i = 0; i < m; ++i) {
            for (int j = 0; j < n; ++j) {
                if (i > 0) grid[i][j] += grid[i-1][j];
                if (j > 0) grid[i][j] += grid[i][j-1];
                if (i > 0 && j > 0) grid[i][j] -= grid[i-1][j-1];

                ans += grid[i][j] <= k;
            }
        }
        return ans;
    }
};
```

**时间复杂度** O(mn)，**空间复杂度** O(1)（原地修改）。

## 方法二：维护每列元素和

### 核心思路

逐行扫描，用 `col_sum[j]` 维护从第 0 行到当前行第 `j` 列的累积和；再用前缀和 `s` 从左到右累加 `col_sum[j]`，得到以 `(0,0)` 为左上角、`(i,j)` 为右下角的子矩阵之和。

一旦 `s > k`，当前行后续列的和只会更大，立即 `break` 跳出内层循环（**早退优化**）。

```cpp
class Solution {
public:
    int countSubmatrices(vector<vector<int>>& grid, int k) {
        int n = grid[0].size();
        vector<int> col_sum(n);
        int ans = 0;

        for (auto& row : grid) {
            int s = 0;
            for (int j = 0; j < n; ++j) {
                col_sum[j] += row[j];
                s += col_sum[j];
                if (s > k) break;   // 后续列的和更大，直接跳过
                ++ans;
            }
        }
        return ans;
    }
};
```

**时间复杂度** 最坏 O(mn)，有早退时更快；**空间复杂度** O(n)。
